# Welcome to This Next.js App (Yes, It's Actually Good, We Promise)

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app). It was carefully crafted by developers who definitely did not just run a CLI command and walk away.

## Getting Started (a.k.a. "Why Isn't It Working?")

First, install your dependencies. This downloads approximately the entire internet into `node_modules/`:

```bash
npm install
```

Then, run the development server. Cross your fingers:

```bash
npm run dev
# or yarn dev, if you're fancy
# or pnpm dev, if you're very fancy
# or bun dev, if you live on the edge
```

Open [http://localhost:3000](http://localhost:3000) with your browser. If it works, congratulations! If it doesn't, have you tried turning it off and on again?

## Editing

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit it — which sounds great until you realize you've introduced a typo and now the entire screen is red at 2am.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font). Geist is a font. It is very clean. Much cleaner than your code, probably.

## Testing

We have tests! Real ones! Run them with:

```bash
npm test
```

They're written with Vitest. Some of them pass. All of them were written with pure intentions and absolutely no copy-pasting from Stack Overflow.

## CI/CD

This project uses GitHub Actions for continuous integration. Every pull request gets checked automatically, because humans cannot be trusted. The workflows live in `.github/workflows/` and they will judge your code so you don't have to.
